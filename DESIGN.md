Design principles
-----------------

- **Keep things simple**

  Introduce complexity only when and if it is necessary. Stick to web
  platform features and avoid unnecessary abstractions.

- **It doesn’t have to look and work the same in every browser**

  Core features MUST be available in all browsers, but experience can be
  enhanced in browsers that support specific features (e.g. view
  transitions, localisation, specific CSS features).
